1. Get Metadata

Endpoint to get metadata about a text file and a substring.

POST /metadata

Parameters:

  • text (string): The content of the text file.
  • substring (string): The substring to search in the text (case-insensitive).

2. Parse URL

Endpoint to parse a URL and provide human-readable information.

POST /parse_url

Parameters:

  • url (string): The URL to parse.

3. Serve Images

Endpoint to serve images based on the provided path.

GET /images/{image_path}

4. Additional Info

Other endpoints:

  • GET /: Main HTML page.
  • GET /info: JSON with additional information.
