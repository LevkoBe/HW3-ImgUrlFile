1. Get Metadata

Endpoint to get metadata about a text file and a substring.

POST /fileParse

Parameters:

  • file (string): The content of the text file.
  • string (string): The substring to search in the text (case-insensitive).

2. Parse URL

Endpoint to parse a URL and provide human-readable information.

GET /urlParse?url=https://example.com GET /urlParse?url=http://127.0.0.1:8000/urlParse?url=url GET /urlParse?url=http://example.com/products/products/products?category=electronics%26page=2

Parameters:

  • url (string): The URL to parse.

3. Serve Images

Endpoint to serve images based on the provided path.

GET /image/img.png
GET /image/img_1.png GET /image/img-2.png GET /image/img_3.png

4. Additional Info

Other endpoints:

  • GET /: Main HTML page.
  • GET /info: JSON with additional information.
